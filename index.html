This Software has been developed to visualize the DDA (Digital Differential Analyzer) algorithm for drawing a line.
The options on the top right corner are:
1) Type: Choose the line equation form.
2) Grid Color: Change the grid lines color.
3) Line Color: Change the line color.
4) Grid Coordinates: Toggle grid coordinates visibility.
5) Line Width: Adjust the line width.
6) Inputs: Set the parameters for line drawing.
7) Zoom: Scroll up/down using your trackpad/mouse
wheel to zoom in/out.
